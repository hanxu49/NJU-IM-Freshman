# NJU-IM-Freshman

## 关于项目
这个项目的创建灵感来源于NJUIM萌新手册，关键词是分享，在此向在萌新手册上书写过自己青春（bushi 的信管同学们致敬。

起因：某天我在群里回答了一个同学的问题，发现他遇到的问题在之前就有几个同学遇到并已经在群里提问过，同时得到了回答，但由于群里消息众多，很多时候我们没有办法去关注到每一条信息，所以当自己遇到一个其实已经被提问并被解决的问题的时候，可能很自然的想法是先在群里提问一下。

正是基于以上的问题，我萌生了一个这样的想法，我们有不少之前在南大本校本科的同学，对南大的很多东西比较熟悉，可以分享一些已有的信息给大家，同时大家可以对一些没有的信息进行提问，然后知道的同学可以来进行回答补充，不断完善。同时，因为每个人获取的信息有限，在你查阅本手册的时候，可能会发现一些你觉得有用但上面没有的东西，你可以进行添加补充完善。

当然，这不仅仅是一个Freshman手册，我觉得更多的是一个分享平台，只要是积极向上正能量的相关内容，都有其值得分享的价值。

Gifts of roses, hand a fragrance.

## 使用指南

关于内容，其实从各个文件夹就已经可以明确看到本仓库所存放的东西。其中需要说明的是，目前我们在部分内容上“模糊”了本科生和研究生之间的界限，一个原因是我们觉得当我们现在重新去审视本科所学的内容时应该会有不同的学习体验，当然另一个原因是我们计划之后也把本科新生相关的内容放进来，但今年南大又推出了新的培养模式——书院制，我们目前尚不清楚具体的细节，只能之后再根据实际情况进行调整。

使用方法上，其实这个不用过多介绍，只需在对应文件夹下寻找相应的文件即可，每个文件对应的内容都已经在标题中明确。

当然，如果你想下载某一个指定的文件，可能需要插件来辅助下载，这里推荐一个我正在用的GitZip for github，可以点击下载Github中的单一文件。详见《使用GitZip for github工具下载仓库中的单个文件.pdf》

最后，已经熟练使用Github的朋友们忽略以上，相信你们已经有很多的仓库食用方法，并欢迎与我们分享~
