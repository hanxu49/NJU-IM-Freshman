# NJU-IM-Freshman


这个项目的创建灵感来源于NJUIM萌新手册，起因是某天我在群里回答了一个同学的问题，发现他遇到的问题在之前就有几个同学遇到并在群里提问过，并得到了回答，但由于群里消息众多，很多时候我们没有办法去关注到每一条信息，所以当自己遇到一个其实已经被提问并被解决的问题的时候，可能很自然的想法是先在群里提问一下。

正是基于以上的问题，我萌生了一个这样的想法，我们有不少之前在南大本校本科的同学，对南大的很多东西比较熟悉，可以分享一些已有的信息给大家，同时大家可以对一些没有的信息进行提问，然后知道的同学可以来进行回答补充，不断完善。同时，在你查阅本手册的时候，可能会发现一些你觉得有用但上面没有的东西，你可以进行添加补充完善。

当然，这不仅仅是一个Freshman手册，我觉得更多的是一个分享平台，只要是积极向上正能量的相关内容，都有其值得分享的价值。

Gifts of roses, hand a fragrance.
